# FitnessOS — Future Features

## Movement Learning & Guidance (Phase 3)

Goal: Help user learn proper form and avoid common mistakes, especially important for returning to lifting after a break.

Planned features:
1. `commonMistakes` field on exercises (text array)
2. `videoUrl` field linking to demo videos
3. `scalingOptions` field for modifications
4. In-app Claude chat for form questions, scoped to current exercise
5. Pre-lift form cues displayed before first set
6. Post-set check-in ("Any discomfort?")
7. Spine-specific warnings for L4/5, S1 considerations (make more prominent)
8. Visual form guides / animations

Priority: After core app is stable on TestFlight
